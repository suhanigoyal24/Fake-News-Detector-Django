Fake News Detector

Django + Django REST Framework + NLP + Machine Learning
A production-style backend application that detects whether a news article is Real or Fake using NLP preprocessing and a trained machine learning model integrated into Django.



Key Highlights
RESTful API built with Django REST Framework
ML model integrated into backend for real-time prediction
NLP preprocessing using spaCy
Confidence score generation
Prediction history stored using Django ORM
Clean modular backend architecture

Tech Stack
Backend: Python, Django, DRF
ML/NLP: scikit-learn, spaCy, TextBlob, joblib
Database: SQLite
Testing: Postman

What This Project Demonstrates
Integrating ML models into production-ready web applications
Designing and validating REST APIs
Backend architecture and data handling
End-to-end deployment-ready workflow
